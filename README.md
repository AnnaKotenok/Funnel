На основе данных использования мобильного приложения для продажи продуктов питания проанализирована воронка продаж, а также оценены результаты A/A/B-тестирования. В данном проекте мной были изучены принципы событийной аналитики. Я построила воронку продаж, исследовала путь пользователей до покупки. Проанализировала результаты A/B-теста введения новых шрифтов. Сравнила 2 контрольных группы между собой, убедилась в правильном разделении трафика, а затем сравнила с тестовой группой Выявлено, что новый шрифт значительно не повлияет на поведение пользователей.
